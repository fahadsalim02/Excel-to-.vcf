# sheets-to-vcf
Sheets to VCF with batch rename feature

# How to use:
* Clone this repo
      
      git clone https://github.com/reenphygeorge/sheets-to-vcf
* Rename your excel file to 'sheet.xlsx' and add it to the project folder
* Install openpyxl
    
      pip install openpyxl
* Now run the code
